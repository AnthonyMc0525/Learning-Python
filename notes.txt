range(from_value, to_value)
from_value is inclusive and to_value is exclusive

dict methods {
  .items() - creates a special list of tuples (usually use list() with this so that we can use the list)
  .get(key, defaults) - gets the value at a key but wont throw an error for non existant value
  .clear() - erases the whole dict
  .copy() - shallow copies a dict
  .keys() - gets all keys in dict
  .values() - get all values from the dict
}
